# 車両重量と終端速度の計算メモ

車両（編成）が実際に出せる最高速度（終端速度）が、積載量を含む重量にどう依存するかをまとめます。

## 前提となる積算値

- **total_weight (W)**: 編成中の全車両の `sum_weight` 合計。`sum_weight = 車両自重 + 積載重量` で、積み下ろしのたびに更新されます。
- **friction_weight (F_f)**: `current_friction × 車両重量` を全車両で合計。カーブ・勾配で `current_friction` が変化します。
- **total_power (P_total)**: 編成全体の出力（ギア補正込み）。
- **speed (v)**: 内部速度単位（約 10/128 km/h）。

## 残余パワー式（終端速度の決定）

終端速度は「残余パワー = 0」になる速度として求められます。コードの式は次の通りです：

```math
\text{residual}(v) = P_{total} - \left( \frac{ v \cdot \left( \frac{F_f \cdot v}{3125} + 1 \right) }{2048} \right) - \frac{ W \cdot 64 }{1000 }
```

- 第 2 項: 速度と摩擦重量に比例する **動的抵抗**（空気抵抗+転がり抵抗の近似）。
- 第 3 項: 重量に比例する **静的抵抗**（基礎的な転がり/勾配抵抗）。
- `residual(v) = 0` となる `v` が終端速度。残余が負なら加速不可、正なら加速余力あり。

## 終端速度の求め方

1. `calc_max_speed()` が **2 分探索**で `residual(v)` が 0 になる速度を探索します（精度 0.5 km/h）
   - 速度上限は線路等の `speed_limit` でクリップ。
2. `res_power()` で残余パワーを評価し、符号反転点を探すことで終端速度を決定。

## 重量・積載が効くポイント

- **積載で W が増える** → 静的抵抗 `W×64/1000` が直線的に増加。
- **積載で F_f も増える** → 動的抵抗項 `v²` に掛かる係数が大きくなり、高速域で効きやすい。
- **カーブ/勾配で current_friction が変化** → `F_f` が増減し、同じ重量でも終端速度が上下。
  - 直線: 基本 1
  - カーブ: 8 に跳ね上がり、大きく減速傾向
  - 上り勾配: `+15×(勾配高さ²)` を加算
  - 下り勾配: `-7×(勾配高さ²)` を加算（抵抗減少）
- **パワー側の上限**: total_power が固定なら、重量増加に応じて終端速度が低下。

## まとめ（実運用の読み替え）

- 編成を重くする（積載量を増やす、または牽引車を減らす）と、
  1. 低速域では静的抵抗増で発進が鈍くなる。
  2. 高速域では動的抵抗係数が増え、到達し得る最高速度が下がる。
- カーブや上り勾配では `current_friction` が大きくなるため、同じ編成でも終端速度がさらに下がる。
- 逆に、空荷や下り勾配では抵抗が減るため、設定最高速度や線路速度制限に近づきやすい。
